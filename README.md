# PezuñoKeep

Gestión de caballos y equipamiento - App web tipo Kanban con PWA.

## 🐴 Características

- **Kanban Board**: Organiza tus tareas en columnas (Inbox, Hoy, Calendario, Proyectos, Referencias)
- **Sistema de prioridades**: Alta, Media, Baja con indicadores visuales
- **Drag & Drop**: Mueve tarjetas entre columnas fácilmente
- **PWA**: Instálala en tu móvil como app nativa
- **Offline**: Funciona sin conexión
- **Recordatorios**: Alerta diaria para medicamentos
- **Iconos personalizados**: 343 iconos de caballos y equipo extraídos del branding

## 🚀 Instalación

### Desde el navegador:
1. Abre la app en tu navegador
2. Click en "Agregar a pantalla de inicio" (Chrome) o "Compartir → Agregar a pantalla de inicio" (Safari)
3. ¡Listo! Funciona como app nativa

### En PC:
1. Abre `index.html` en tu navegador
2. O usa un servidor local: `python -m http.server 8080`
